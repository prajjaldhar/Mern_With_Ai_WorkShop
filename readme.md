## 📖 Overview

**MERN With AI Workshop** is a hands-on, industry-driven learning experience designed to help developers build **AI-powered full-stack applications** using the MERN stack.

This repository focuses on combining traditional web development with modern AI capabilities to create **intelligent, scalable, and production-ready systems**.

---

## 🎯 Objectives

- Build a strong foundation in **MERN Stack architecture**
- Integrate **AI APIs (LLMs)** into real-world applications
- Develop **intelligent backend services and APIs**
- Enhance frontend with **AI-driven user experiences**
- Follow **industry-level development practices**

## 🛠️ Installation & Setup

### 1. Clone Repository

```bash
git clone https://github.com/prajjaldhar/Mern_With_Ai_WorkShop.git
cd Mern_With_Ai_WorkShop
```

### 2. Install Dependencies

#### Server

```bash
cd server
npm install
```

### 3. Configure Environment Variables

Create a `.env` file inside `server/`:

```env
PORT=5000
MONGO_URI=your_mongodb_connection
Gemini_API_KEY=your_api_key
```

---

## ▶️ Run the Application

- Install Live Server
- Run the Html

### Start Backend

```bash
npm run dev
```
